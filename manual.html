Operator Guide

One place to see what is active, where it lives, and how to resume it.

Project Dashboard is a local-first command center for `/srv/cloud-mirror`. It keeps a validated SQLite project database, requires API authorization, limits browser writes to state changes and explicit selected-project syncs, and keeps every AI call behind server-side guardrails.

Golden Rule

Use `project-dashboard-ingest` or dashboard APIs for updates. `data/projects.json` is only a diagnostic sentinel.

Block Diagram

Request and Data Flow

Feature

Project List

Cards show state, summary, path, next action, blockers, and update time. State toggles filter the portfolio without changing project data.

Feature

Command Center

The selected-project panel keeps the work surface focused: next moves, blockers, time estimate, handoff path, and action buttons.

Feature

Resume Helpers

ChatGPT copies a context packet. Codex copies an SSH command that resumes the latest matching server session. Terminal opens an SSH shell in the project folder.

Feature

AI Sync

Allowed browser users can run one selected-project sync from the Command Center. The server still requires explicit user-action headers, budget caps, daily call caps, cooldowns, and database verification.

Reference

API Surface

Route Purpose Write Risk
`GET /api/dashboard` Returns the authenticated SQLite-backed dashboard payload used by desktop and mobile. None; API auth required.
`GET /api/health` Reports SQLite availability, project count, schema version, and generated timestamp. None; API auth required.
`GET /api/ai-budget` Reads local spend metadata and guardrail state. None; API auth required.
`GET /api/codex-resume-command` Finds the newest matching `codex resume` command from the server resume log. None; API auth required.
`GET /api/state-audit` Returns recent project-state changes from the SQLite audit table. None; API auth required.
`POST /api/project-state` Requires allowed Cloudflare Access email or server-only write token, then changes one selected project's state. Writes SQLite after validation, backup, audit, and verification.
`POST /api/sync-selected-project` Requires allowed Cloudflare Access email or server-only write token, then runs one guarded OpenAI sync for one selected project. Writes SQLite after validation, backup, audit, and verification.
